This project provides a comprehensive analysis of the AdventureWorks sales data using Power BI and DAX measures. The focus is on deriving meaningful insights through interactive dashboards and calculated measures from the provided dataset.

Data Source
Sales.xlsx: Contains the sales data used for analysis.
Data Modeling
Product Hierarchy: A structured model to analyze sales performance across different product levels (Category, SubCategory, Product).
DAX Measures
# Orders: Count of total orders.
# Order Details: Count of order details.
Total SubTotal: Aggregated subtotals of all orders.
Total Tax: Total tax amount for orders.
Total Freight: Total freight cost for orders.
Total Due: Total due amount for all orders.
